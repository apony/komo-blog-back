# komo-blog-back
Blog Backend Project【Koa Server and Mongodb Database】
>技术栈：koa+mongodb

## 介绍
+ 此项目为reant-blog-front提供接口服务

## 安装依赖
```shell
npm install
```

## 运行服务
```shell
npm run server
```

## tips
+ 运行端口：http://loaclhost:8000
+ 安装mongodb，q群有安装教程(723594858)
+ mongoDB可视化工具：Robo3T
+ 修改Robo3T连接端口8000，默认为27017（安装目录找到mongod.cfg修改port字段即可，然后重启MongoDB服务）
